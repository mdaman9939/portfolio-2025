Case Study

CRM / Web App

CRM Grievance Management System

Overview

A public grievance management CRM that lets citizens raise complaints and lets staff track and resolve them at scale.

The Challenge

Complaints were arriving through scattered channels with no central tracking, leading to lost requests and slow resolution times.

The Solution

I built a scalable CRM with ticketing, status tracking and an admin panel using PHP, MySQL and Bootstrap, with a clean public-facing intake form.

Key Features

  • Public complaint intake
  • Ticket status tracking
  • Admin resolution panel
  • Searchable records

Tech Stack

Bootstrap PHP MySQL REST APIs

Outcome

A single source of truth for grievances — nothing falls through the cracks, and resolution times dropped noticeably.
